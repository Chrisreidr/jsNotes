Strings

Strings are primitives and impossbile to mutate. The results of any methods used will have to be stored in a variable or data structure. These methods can be called on one another as long as the result of the previous method is a string.

String Methods

Example: const str = "Chris likes programming!";

  • str.length // 24 - Used to find the amount of characters in a string.
  • str.indexOf('r') // 2 - Returns the position of the first occurence of a certain letter or entire word in the string. Keep in mind strings are 0-based and spaces count as a position as well. Entries are case-sentitive.
  • str.lastIndexOf('r') // 16 - Similar to indexOf() except it returns the position of the last occurence of the stated character or entire word.
  • str.slice(0) // 'C' - Takes one parameter to extract the string beginning at that position or a beginning and end paramter to extract a portion of characters.
  • str.toLowerCase() // 'chris likes programming!' - Takes no arguements and will convert the string it is called on to lowercase letters. Can be called on a variable containing a string or on a string itself.
  • str.toUpperCase() // 'CHRIS LIKES PROGRAMMING!' - Takes no arguements and will convert the string it is called on to uppercase letters. Can be called on a variable containing a string or on a string itself.
  • str.trim() // "Chris likes programming!" - Used to remove white space at either end of a string. There is also .trimStart() and .trimEnd()
  • str.replace('!', '?') // "Chris likes programming?" - Used to replace one string (first argument) with a sceond string (second argument). Only replaces the first occurence of the string. To replace each occurence use str.replaceAll(). Another example includes str.replace(str[0], str[0].toLowerCase()) // "chris likes programming!"
  • str.repeat(3) // "Chris likes programming!Chris likes programming!Chris likes programming!" - Simply takes the number of times you would like to repeat as an argument.
  • str.padStart(30, '+') // "++++++Chris likes programming!" - Adds a number of characters to the string until the string has a certain desired length. The first arguement is the length of the entire string and the second argument is the character used to pad. We can also pad the end by adding str.padStart(30, '+').padEnd(36, '+') // "++++++Chris likes programming!++++++"
  • str.split(' ') // ['Chris', 'likes', 'programming!']; - Will return an array of elements seperated by the specified argument. Which in this case is a space.
  • ['Chris', 'likes', 'programming!'].join(' ') // "Chris likes programming!"; - The inverse of the .split() method, it will join an array of elements into a string that is separated by the argument passed. Which in this case is a space.

    • Boolean Methods

    Used whenever we need to make a decision based on the contents of a string, these methods are useful.

  • str.inludes('likes') // true - Will return true or false in the string includes the specified aurgument. Is case-sensitive.
  • str.startsWith('Chris') // true - Will return true or false in the string starts with the specified aurgument. Is case-sensitive.
  • str.endsWith('Chris') // false - Will return true or false in the string ends with the specified aurgument. Is case-sensitive.

    • Use-case

    A great use-case for these indexs usually occures when needing to etxract a string through specified positions.

  • str.slice(6) // likes programming! - The slice method needs indexes as arguements which dictates where the extraction will start. The result "likes programming" is referred to as a sub-string. But does not change the initial string.
  • str.slice(6, 11) // likes - We can also specify and ending parameter. The method will stop extracting just before the specified end parameter.
    Note: the length of the extracted string will always be the specified end minus the beginning parameter. Meaning the length in this case is 5

    Last word - str.slice(str.lastIndexOf(' '));

  • str.slice(-1) // ! - We can also specify negative values. They will start counting backwards.
  • str.slice(-11, -1) // programming - Can also specify a negative beginning ang ending value.
